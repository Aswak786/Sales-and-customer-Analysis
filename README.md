# Sales-and-customer-Analysis(Power BI)

# Project Overview:

This Power BI project provides an in-depth analysis of sales performance and customer behavior, helping businesses track key revenue metrics, product performance, and shipping preferences. It enables decision-makers to understand trends, identify profitable customers, and optimize operations.

# Objectives

1. Evaluate total sales, profit, quantity sold, and average discounts.

2. Track monthly sales trends and customer-wise profitability.

3. Analyze sales distribution across segments, products, and ship modes.

4. Provide actionable insights for strategic business planning.

# Tools & Technologies

1. Power BI Desktop

2. DAX (Data Analysis Expressions)

3. Power Query Editor

4. Excel / CSV Sales Dataset

# Key Features

# KPIs:

1. Total Sales, Total Profit, Total Quantity, Average Discount.

2. Interactive Filters: Segment, Order Date, and Ship Mode.

# Visuals:

1. Line chart for monthly sales trend.

2. Bar charts for sales by Product ID and by Ship Mode.

3. Combined chart for Sales vs. Profit comparison.

4. Table view for Customer-wise sales and profit.

5. Dynamic Segmentation: Consumer, Corporate, and Home Office segments.

6. Clean Dashboard Layout: Easy-to-read cards, charts, and filters for quick insights.

# Insights Generated

1. Achieved total sales of 1 billion with matching profitability in the analyzed period.

2. Standard shipping mode contributes most of the overall sales.

3. Top-performing products (e.g., Furniture and Office Supplies) drive a major portion of revenue.

4. Identified high-value customers and loss-making segments, supporting profit optimization.



# How to Use

1. Download or clone this repository.

2. Open the .pbix file in Power BI Desktop.

3. Load or refresh the dataset in Power BI.

5. Explore interactive visuals and filters to gain insights.
